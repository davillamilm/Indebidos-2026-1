# Indebidos-2026-1
Este repositorio muestra el avance de los proyectos del grupo conformado por Juan Jose Gonzalez, Esteban Trejo y David Villamil.

Proyectos en este repositorio:

- Entrenamiento de un IA por medio de Google colab.
- Desarrollo de un microtrolador utilizando el chip de la ESP32-S3.
- Instalacion de un sistema operativo, uso de los pines GPIO, Bluethoot y WiFi.
- Entrenamiento de una IA en el microprocesador Raspberry Pi Zero 2W.
- Conexion entre el microcontralador diseñado y del microprocesador.
